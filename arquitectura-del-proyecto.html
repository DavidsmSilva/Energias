✨ Características Principales

📊 Dashboard (Energía)

Panel de Empresas:
  • Encabezado verde con título "Empresas" y botón "+" para agregar
  • Formulario para crear empresa (solo nombre)
  • Lista de empresas con icono, nombre y menú de opciones
  • Dropdown con opciones: Inventario, Consumo
Resumen de Empresas:
  • Tarjetas con contadores de cada empresa
  • Al hacer clic: filtra tablas y muestra tabs Inventario/Consumo
  • Filtro por tipo de energía específico
  • Modal de detalle con campos existentes

📈 Inventario y Consumo

  • Sin empresa: Tabla con todos los registros
  • Con empresa: Formulario + tabla específica
  • Inventario: tabla verde | Consumo: tabla azul

🚀 Comandos

ng test Ejecutar pruebas
ng lint Verificar código

📝 Notas del Desarrollador

  • AppComponent es standalone (Angular 15+)
  • Los filtros de tipo no tienen opción "Todos"
  • Panel de empresas integrado en Dashboard
  • Los modales muestran solo campos con valor
  • Uso de RxJS para manejo reactivo de datos
  • Angular Material para componentes UI
