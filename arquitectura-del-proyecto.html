Arquitectura del Proyecto - Energia Inventory

Este documento describe la estructura, frameworks y tecnologías utilizadas en el proyecto.

Frameworks y Tecnologías

  • Angular 17 - Framework principal para el frontend
  • TypeScript - Lenguaje de programación
  • Angular Material - Biblioteca de componentes UI
  • RxJS - Programación reactiva para manejo de observables
  • HTML5/CSS3 - Estructura y estilos
  • Node.js - Entorno de ejecución
  • npm - Gestor de paquetes

Estructura del Proyecto

  • src/app/ - Código principal de la aplicación
  • src/app/components/ - Componentes reutilizables (header)
  • src/app/pages/ - Páginas principales (dashboard, inventario, consumo, ajustes)
  • src/app/services/ - Servicios para gestión de datos
  • src/app/shared/ - Componentes compartidos
  • src/app/models/ - Modelos de datos
  • src/app/empresa-list/ - Componente de lista de empresas

Componentes Principales

  • AppComponent - Componente raíz de la aplicación con menú lateral (standalone)
  • HeaderComponent - Barra de encabezado con título dinámico
  • DashboardComponent - Página de inicio (Energía) con panel de empresas y resumen
  • InventarioComponent - Gestión de inventario por empresa
  • ConsumoComponent - Gestión de consumo por empresa
  • AjustesComponent - Página de ajustes
  • EmpresaListComponent - Componente de lista de empresas
  • SlidesDialogComponent - Diálogo para diapositivas

Servicios

  • EmpresaService - Gestión de empresas (listado, selección, agregar)
  • DatosService - Manejo de datos de inventario y consumo

Modelos de Datos

  • Empresa - id, nombre
  • RegistroInventario - Tipos: electrica, renovable, combustibles, biomasa_electrica, biomasa_calor
  • RegistroConsumo - Tipos: gas_electrica, gas_calor, vendida, recibida, cedida

Funcionalidades Principales

Menú Lateral (AppComponent)

  • Navegación: Energia (🏠), Ajustes (⚙️)
  • Encabezado: ENERGÍA

Página de Energía (Dashboard)

Panel de Empresas:
  • Encabezado verde con título "Empresas" (🏢) y botón "+" para agregar
  • Formulario para crear empresa (solo nombre)
  • Lista de empresas con icono, nombre y menú de 3 puntitos (⋮)
  • Dropdown con opciones: Inventario, Consumo
  • Vista de resumen de todas las empresas (tarjetas con contadores)
  • Al hacer clic en una empresa:
    • Se filtran las tablas para mostrar solo esa empresa
    • Se muestran tabs: Inventario | Consumo
    • Tablas con Tipo y Fecha
    • Dropdown para filtrar por tipo de energía específico (sin opción "Todos")
    • Al hacer clic en una fila se abre modal con detalles completos
  • Modal de detalle muestra solo los campos del formulario seleccionado (no campos vacíos)

Inventario y Consumo

  • Sin empresa seleccionada: Tabla con todos los registros de todas las empresas
  • Con empresa seleccionada (/inventario/:id o /consumo/:id): Formulario + tabla de esa empresa
  • Formulario primero, tabla debajo
  • Inventario: tabla verde | Consumo: tabla azul

Tipos de Formularios

  • Inventario:
    • Energía eléctrica del servicio público
    • Energías renovables
    • Energía a partir de combustibles
    • Biomasa para energía eléctrica
    • Biomasa para generar calor
  • Consumo:
    • Gas para energía eléctrica
    • Gas para generar calor
    • Energía Vendida
    • Energía Recibida
    • Energía Cedida

Rutas

  • / - Dashboard (Inicio - Energía)
  • /inventario - Inventario de todas las empresas
  • /inventario/:empresaId - Inventario de una empresa específica
  • /consumo - Consumo de todas las empresas
  • /consumo/:empresaId - Consumo de una empresa específica
  • /ajustes - Página de ajustes

Comandos para Ejecutar el Proyecto

  • npm install - Instalar dependencias
  • npm start - Iniciar servidor de desarrollo (puerto 4200)
  • ng serve --port [puerto] - Iniciar en puerto específico
  • npm run build - Construir para producción

Notas del Desarrollador

  • AppComponent es standalone (Angular 17+)
  • Los filtros de tipo en el resumen de empresas no tienen opción "Todos" - siempre hay un tipo seleccionado
  • El panel de empresas está integrado en la página de Energía (Dashboard)
  • Los modales muestran solo los campos que tienen valor registrado
