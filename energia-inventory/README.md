# Energia Inventory - Frontend Angular App (read-only MVP)

Instrucciones rápidas para arrancar desde tu máquina local:
- Asegúrate de tener Node.js y npm instalados.
- Abre una terminal y ejecuta: cd "C:\Users\T.I\OneDrive\OneDrive - FRACTAL ESTRATEGIAS SOSTENIBLES SAS\Escritorio\Empresa-Energia\energia-inventory"
- Luego: npm install --legacy-peer-deps
- Finalmente: npm start

Rápido para Windows: usa el script run-energy-inventory.bat ubicado en la carpeta raiz del proyecto (energia-inventory/run-energy-inventory.bat). Doble clic o ejecútalo desde la consola para instalar y arrancar.
